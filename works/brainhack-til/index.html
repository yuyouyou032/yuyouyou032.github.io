Object Re-identification

A machine learning project, part of 2023 DSTA Brainhack-TIL competition

Summary

In 2023, I teamed up with a few friends from HCI Infocomm & Robotics society to participate in the DSTA Brainhack-TIL Machine Learning & Robotics challenge open category.

The machine learning section is mainly constituted by two parts: audio identification and computer vision.

I was mainly responsible for the computer vision component, which requires our model to be able to draw a bounding box around an object and identify whether it is of the same type as any other objects existing in a database.
